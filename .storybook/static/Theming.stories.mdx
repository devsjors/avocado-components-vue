# Avocado Media - Design System - Vue

## Recommended folder structure
```js
theme
└───ThemeName
    └───index.js
        Cores
        └───index.js
        └───Colors
        │   └───index.js
        │   └───primary.js
        │   └───secondary.js
        │   └───base.js
        │   └───feedback.js
        └───Typography
        │   └───index.js
        │   └───titles.js
        │   └───subtitles.js
        │   └───text.js
        └───Spacing
        │   └───index.js
        Atoms
        └───index.js
        └───Button
        │   └───index.js
        └───Link
            └───index.js
```

## ThemeName

```js
// theme / ThemeName / index.js
import cores from "./Cores";
import atoms from "./Atoms";

const CustomTheme = {
  cores,
  atoms,
};

export default CustomTheme;
```

## Cores

```js
// theme / ThemeName / Cores / index.js
import colors from "./Colors";
import typography from "./Typography";
import spacing from "./Spacing";

const cores = {
  colors,
  typography,
  spacing,
};

export default cores;

```
